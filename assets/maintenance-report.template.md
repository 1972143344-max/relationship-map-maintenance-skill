# Maintenance Threshold

- Thresholds:
  - split or archive older maintenance reports when one report grows past `180` lines or starts covering multiple unrelated maintenance runs
  - if a report can no longer surface the main findings near the top, treat it as oversized even before the line threshold
- Trigger:
  - if this report is over threshold or too noisy for one-pass review, explicitly tell the user that maintenance-report cleanup is needed and propose archival or semantic splitting
